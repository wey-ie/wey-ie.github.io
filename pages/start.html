快速入门 ​

本篇将向您介绍JoUI的入门方法，包含引擎配置、初始化、创建窗口等。

开始 ​

以Visual Studio 2022 C++项目为例。

将从开始-下载中下载的压缩包解压，将获得一个“JOUI”文件夹，进入该文件夹，可以看到以下文件：

文件结构：

JOUI
  │
  ├ E\：易语言模块、Demo、资源文件等
  │
  ├ msvc\：C++案例项目、资源文件等
  │
  ├ libJoui@X64.dll：64位动态库文件
  │
  ├ libJoui@X86.dll：32位动态库文件
  │
  └ 更新.txt：引擎库更新日志文件

注意：若您得到的不是上述文件结构，则可能下载文件不完整或为其它三方版本，请从官方通道重新下载。

我们需要用到的为“msvc”文件夹和“libJoui@*.dll”文件，具体位数根据您的需要选择即可，我们以32位为例。

一、创建项目 ​

首先，打开VS，选择“创建新项目”，随后进入下方页面：

① 项目语言根据您所用语言选择，这里以C++为例；
② 项目平台选择“Windows”；
③/④ 项目类型可选择“空项目”或“Widnows桌面应用程序”，选择“空项目”后面会多出部分配置步骤；
⑤ 选择完成后单击下一步，编辑项目参数以创建项目。

二、项目配置 ​

以下配置可能默认已相符，但请您手动检查以避免不必要的错误。

在修改配置前，请先将程序位数修改为您选择的位数，后续若修改位数也需再次执行以下设置；单击菜单“项目-*属性”，进入属性设置页。

提示

本步骤中第3、5、6条为必须设置，否则可能导致程序无法编译启动；其余为建议设置，可根据需求自行选择设置（部分情况下错误的设置也可能导致程序无法编译启动）。

  1. C/C++ - 常规：
    SDL检查：否(/sdl-)
    多处理器编译：是(/MP)

  2. C/C++ - 优化：
    优化（不要求完全一致）：优化(优选速度)(/Ox)
    启用内部函数：是(/Oi)

  3. C/C++ - 预处理器
    预处理器定义：不要求完全一致，但如果包含有_CONSOLE;，请去掉

  4. C/C++ - 代码生成：
    基本运行时检查：默认值
    运行库：多线程调试(/MTd)

  5. C/C++ - 高级：
    调用约定：_stdcall(/Gz)

  6. 链接器 - 系统
    子系统：窗口(/SUBSYSTEM:WINDOWS)

三、代码 ​

为规范操作，若您创建项目时选择的是“空项目”，请手动创建与项目名一致的.h和.cpp文件，这里以“jouidocsdemo”为例。

在开始写代码前，需先将JOUI文件夹中的libJoui@\*.dll、msvc\lib\libJoui@\*.lib、msvc\Theme.zip三个文件以及msvc\include文件夹复制到项目目录下（*为您选择的动态库位数，32位为86，64位为64）。

首先在jouidocsdemo.h（此处应为您创建的项目名，此后不再复述）头文件中添加以下代码以引用库：

CPP
#include <Windows.h>
#include <string>
#include "include/libJoui_Main.h"

#pragam comment(lib,"libJoui@X86.lib")

随后在jouidocsdemo.cpp文件中引用该头文件（“Windows桌面应用程序”已自动引用）：

CPP
#include "jouidocsdemo.h"

using namespace std;//方便调用std命名空间的函数

若您创建时选择的是“Windows桌面应用程序”，请删掉jouidocsdemo.cpp文件中除以上引用外的所有代码。

接下来需要定义程序的入口函数（函数规范请参阅微软WinMain：应用程序入口点文档）：

CPP
int WINAPI wWinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, PWSTR pCmdLine, int nCmdShow);

要使用引擎库，需先初始化引擎，也需在结束时卸载引擎，对应函数：

CPP
//初始化引擎，在程序运行起始位置
void Jo_Init(H_INITINFO *lpConfigInfo = nullptr);

//卸载引擎，在程序运行结束位置
void Jo_UnInit();

初始化引擎完成后，创建窗口需先注册窗口类名，创建后需绑定窗口，对应函数：

CPP
//第一步：注册窗口类名
WORD Jo_WndRegisterClass(LPCWSTR lpwzClassName, HICON hIcon = NULL, HICON hIconsm = NULL);

//第二步：创建窗口
HWND Jo_WndCreate(HWND hWndParent, LPCWSTR lpwzClassName, LPCWSTR lpwzWindowName, INT x, INT y, INT width, INT height, INT dwStyle = NULL, INT dwStyleEx = NULL);

//第三步：绑定窗口
HJOGUI Jo_BindWindow(HWND hWnd, HZIPSKIN hTheme, DWORD dwStyle, LPARAM lParam = 0, WinMsgPROC lpfnMsgProc = 0);

注意：创建窗口及后续创建控件相关的部分建议单独定义类或函数进行，而不是放在入口函数内。

最后，我们只需要使用显示窗口函数和窗口消息循环函数，一个空白窗口就完成了：

CPP
//显示窗口
BOOL Jo_ShowWindow(HJOGUI Wndui, INT nCmdShow);

//窗口消息循环
WPARAM Jo_WndMsgLoop();

创建窗口的完整示例代码：

CPP
#include "jouidocsdemo.h"

using namespace std;

HJOGUI m_hObjWndUI = 0;
HWND m_hWnd = 0;

int WINAPI wWinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, PWSTR pCmdLine, int nCmdShow)
{

    H_INITINFO lpConfigInfo{};
    lpConfigInfo.hThemeKey = "JOUI@LIB";
    Jo_ReadFile(L"Theme.zip", &lpConfigInfo.hTheme, &lpConfigInfo.hThemeLen);
    Jo_Init(&lpConfigInfo);
    
    Jo_WndRegisterClass(L"demo_class", 0, 0);
    m_hWnd = Jo_WndCreate(0, L"demo_class", L"Joui_demo", 0, 0, 1080, 600, 0, 0);

    if (m_hWnd != 0)
      {
        m_hObjWndUI = Jo_BindWindowEx(m_hWnd, 0, EWS_MAINWINDOW | EWS_BUTTON_CLOSE | EWS_BUTTON_MIN | EWS_BUTTON_MAX | EWS_MOVEABLE | EWS_CENTERWINDOW | EWS_ESCEXIT | EWS_TITLE | EWS_SIZEABLE | EWS_HASICON);

        //由于某些奇怪的特性，窗口必须设置背景，否则窗口背景会透明
        Jo_UISetLong(m_hObjWndUI, EWL_CRBKG, ARGB(255, 255, 255, 255));

        Jo_ShowWindow(m_hObjWndUI, SW_SHOWNORMAL);
      }

    Jo_WndMsgLoop();
    Jo_UnInit();
    return 0;
}

若您想尝试创建更多组件，请查阅控件目录页面。
